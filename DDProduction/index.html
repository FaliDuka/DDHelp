Welcome toD&D Helper+



This is a Dungeons & Dragons website to help players play the game in a more organized fashion allowing ease of management with the game, while not taking away the players-imagination.

Please note that this is not an actual working website! It is just a prototype designed for CSC355-10

The navigation bar at the top of the page allows a user to browse the webpage, create an account, Create character sheets, and view recent characters. In order to build a "character sheet", the user must create and verify there account in order to properly login and start creating.

Enjoy your "Dungeons & Dragons" experience!
